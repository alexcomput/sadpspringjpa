Módulo Adminitração do Portal Secad

Objetivo

Realizar o gerenciamento e inclusão de informações nas tabelas utilizadas pelo Portal da Secad, por exemplo:
	Gerenciamento de notícias;
	Editais;
	Artigos;
	Decretos;
	Gerenciamento de layout(menu,destaques);
	Gerenciamento de usuarios do Portal.

Este gerenciamento é realizado pelo gestor do Portal, atualmente Rildo.
